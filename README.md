### class for work with smm api
https://justanotherpanel.com

add this to composer.json and run `composer require backendidsiapps/smm-api"`
```
   "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/backendidsiapps/smm-api"
        }
    ],

```
add to .env
```
SMM_API_KEY=YOURAPIKEY
```
